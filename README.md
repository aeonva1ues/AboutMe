## Setup
__1. Clone this repository__
```
git clone https://github.com/
cd 
```
__2. Activate virtual enviroment and install dependencies:__
```
.\venv\Scripts\activate
pip install -r requirements.txt
```
__3. Runserver__
```
cd aboutme
python manage.py runserver
```
